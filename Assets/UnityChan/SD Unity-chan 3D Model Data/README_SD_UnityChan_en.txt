How to use:
1. create a new project with Unity 6000.0.62f1 or later and install "Unity Toon Shader" package from the package manager in advance.
   Details on Unity Toon Shader and its installation can be viewed in its manual:
   https://docs.unity3d.com/Packages/com.unity.toonshader@latest

2. Drag and drop this Unitypackage asset into the project window.  
   Verify that the extraction completed successfully and that a new UnityChan folder has been created inside the Assets directory.

3. Open one of the following sample scenes: 
   - Assets/UnityChan/SD Unity-chan 3D Model Data/Scenes/SD_unitychan_Generic/SD_unitychan_Generic.unity
   - Assets/UnityChan/SD Unity-chan 3D Model Data/Scenes/SD_unitychan_Humanoid/SD_unitychan_Humanoid.unity
   
4. Press the Play button

-------------------------------------------------------------------------------

Operations:
- Alt + Left Mouse: Orbit around SD Unity-Chan
- Alt + Ctrl + Left Mouse: Zoom in/out
- Mouse Wheel: Zoom in/out
- R: Reset view

UI:
- Lock Face: Keeps the currently selected face, even if the main animation changes it
- Random: Randomizes the body animation
- Wind: Toggles wind effects that influence hair via the SpringManager component
- Look At IK (Humanoid only): Makes SD Unity-Chan look at the target
