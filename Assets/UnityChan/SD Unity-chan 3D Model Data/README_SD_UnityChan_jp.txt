SDユニティちゃん 3Dモデルデータ
Unity Technologies Japan

本デジタルアセットデータは「ユニティちゃんライセンス条項（UCL）」（配布時点での最新版は、UCL 3.0）に基づき公開されるデジタルアセットデータです。
最新版の「ユニティちゃんライセンス条項」は以下をご確認ください。

* ガイドライン: https://unity-chan.com/contents/guideline/
* ユニティちゃんライセンス条項: https://unity-chan.com/contents/license_jp/

FAQ　では、本ライセンスの元で、クリエイターの皆さんが「できること」「できないこと」を詳しく具体例をあげて説明しています。
ご使用前に必ず　FAQ　をご一読ください。

-------------------------------------------------------------------------------

【使い方】
1. Unity 6000.0.62f1 以降で新規プロジェクトを作成し、パッケージマネージャーから「Unity Toon Shader」パッケージをインストールしてください。  
   Unity Toon Shader およびそのインストール手順の詳細は以下のマニュアルをご参照ください：  
   https://docs.unity3d.com/Packages/com.unity.toonshader@latest

2. プロジェクト・ウィンドウに、この Unitypackage のアセットをドラッグ＆ドロップします。  
   正常に解凍され、 Assets フォルダ以下に UnityChan フォルダが新規作成されることをご確認ください。

3. 次のいずれかのサンプルシーンを開きます： 
   - Assets/UnityChan/SD Unity-chan 3D Model Data/Scenes/SD_unitychan_Generic/SD_unitychan_Generic.unity
   - Assets/UnityChan/SD Unity-chan 3D Model Data/Scenes/SD_unitychan_Humanoid/SD_unitychan_Humanoid.unity
   
4. 再生ボタンを押します。


-------------------------------------------------------------------------------

操作:
- Alt + 左クリック: SD Unity-Chan の周りを回転
- Alt + Ctrl + 左クリック: ズームイン/アウト
- マウスホイール: ズームイン/アウト
- R: ビューをリセット

UI:
- Lock Face: メインアニメーションで変更されても、現在選択中の表情を維持する
- Random: ボディ・アニメーションをランダムで再生する
- Wind: 風のオン・オフの切り替え。 オンの場合、SpringManager コンポーネントによって髪が揺れるようにする
- Look At IK　（Humanoid のみ）: SD Unity-Chan がターゲットを見るようにする
